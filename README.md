## Deskripsi Project
Project ini merupakan aplikasi dashboard sederhana untuk mahasiswa yang menampilkan data produk, penjualan, dan pendapatan.  

- **Halaman Login:** Mahasiswa dapat masuk menggunakan email sebagai username dan NIM sebagai password. Terdapat validasi input untuk memastikan email dan NIM tidak kosong.  
- **Halaman Dashboard:** Menampilkan ringkasan jumlah total produk, total sales, dan total revenue dalam bentuk card interaktif. Sidebar menampilkan navigasi untuk berpindah ke halaman produk.  
- **Halaman Produk:** Menampilkan data produk dalam tabel interaktif dengan opsi edit dan delete. Terdapat tombol untuk menambahkan produk baru dan navigasi kembali ke dashboard.  

Aplikasi ini sepenuhnya menggunakan **HTML, CSS, dan JavaScript** tanpa backend, sehingga semua data disimulasikan di sisi klien.
